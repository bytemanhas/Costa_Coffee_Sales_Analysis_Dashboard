# Costa_Coffee_Sales_Analysis_Dashboard
Interactive Excel dashboard analyzing Costa Coffee sales performance using Pivot Tables, Pivot Charts, Slicers, and business intelligence storytelling to uncover revenue, product, location, and payment insights.

Project Overview

This project is an interactive Excel Sales Dashboard built for analyzing Costa Coffee’s sales performance across products, locations, time periods, and payment methods. The dashboard provides business insights through KPI cards, pivot charts, slicers, and interactive visualizations.

The objective of this project is to transform raw transactional sales data into meaningful business insights that help decision-makers monitor performance and identify growth opportunities.

Business Problem

Costa Coffee operates across multiple store locations with various product categories and payment methods. The business requires a centralized dashboard to monitor:

Revenue performance
Product sales trends
Store performance
Peak sales hours
Customer payment preferences

Without a dashboard, identifying trends and making data-driven decisions becomes difficult.

Project Objectives
Analyze overall sales performance
Track total transactions and quantity sold
Identify peak sales hours
Evaluate category-wise performance
Identify top-selling products
Compare revenue across store locations
Understand payment mode distribution
